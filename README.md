# AWS Cloud Practitioner Flashcards

Static flashcard site for AWS Certified Cloud Practitioner CLF-C02 revision.

## Live Site

After GitHub Pages is enabled, the site will be available at:

https://ohm-kel.github.io/aws-cloud-practitioner-flashcards/

## Features

- Flip-card active recall
- Domain filters
- Search
- Shuffle mode
- Known / Needs review progress tracking
- Rapid revision filter
- High-yield confused-pair tables

## GitHub Pages Setup

1. Open this repository on GitHub.
2. Go to Settings > Pages.
3. Under Build and deployment, choose Deploy from a branch.
4. Select branch: main.
5. Select folder: /root.
6. Save.

The app stores study progress in the browser with localStorage.
