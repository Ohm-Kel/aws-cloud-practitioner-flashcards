# AWS Cloud Practitioner Flashcards

Static flashcard site for AWS Certified Cloud Practitioner CLF-C02 revision.

## Open Locally

Open `index.html` in a browser.

## Host on GitHub Pages

1. Create a GitHub repository.
2. Add `index.html`, `styles.css`, `app.js`, and this `README.md`.
3. In GitHub, go to **Settings → Pages**.
4. Set **Source** to your main branch and root folder.
5. Save. GitHub will give you a public Pages URL.

## Files

- `index.html` - page structure
- `styles.css` - layout and responsive design
- `app.js` - flashcards, filters, search, progress, and confused-pair tables

The app stores study progress in the browser with `localStorage`.
